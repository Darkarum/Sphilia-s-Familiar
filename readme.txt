Sphilia's Familiar

------------------------------------------------------------

Thank you for purchasing
"Sphilia's Familiar."

This game is made using "RPG Maker VXAce."
To play the game you will need "RPG Maker VXAce RTP."
You can get "RPG Maker VXAce RTP" from the following site.
https://tkool.jp/support/download/rpgvxace/rtp

And the application may not function unless a Japanese language pack is set properly in your PC / the System Locale is set to Japanese. For more details, please refer to the following site.
https://www.dlsite.com/ecchi-eng/faq/detail/=/type/user/mid/7/did/296

Please look at the "Quick Tips" on how to play the game.

Other notable game mechanics include...

*Press the select button during battle to speed up.
*Skip messages with Ctrl.
*Past events can be viewed in specific spots in the game.
*Toggle screen size with F5.

And that would be all.
